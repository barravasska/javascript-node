# 🟢 Node.js on WSL2 Environment

Repository ini berisi catatan dan latihan pengembangan aplikasi **Node.js** di dalam lingkungan **Windows Subsystem for Linux (WSL2)**.

## 🚀 Persiapan Lingkungan (Setup)

Untuk performa terbaik, Node.js diinstal menggunakan **NVM (Node Version Manager)** agar versi Node mudah dikelola.

### 1. Install NVM
Jalankan perintah ini di terminal WSL:
```bash
curl -o- [https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh](https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh) | bash
